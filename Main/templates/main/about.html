Добро пожаловать в "Лапки и кофе" — уютное кото-кафе, а также мой личный полноценный проект с использованием веб-фреймворка Django Rest Framework.

Данный веб-ресурс в первую очередь разработан для онлайн-продажи билетов в кото-кафе, обеспечивающий удобство бронирования для будущих посетителей. Помимо этого сайт предоставляет и другой функционал, такой как оформление опекунства (в формате подписки), запись на взятие в семью понравившегося котика, оформление заявки на передачу пушистика в кафе, просмотр информации обо всех имеющихся котиках и самом заведении, а самое главное - регистрация и авторизация пользователей, включающие личный кабинет с информацией об аккаунте и возможностью ее изменить. На сайте реализована оплата покупок - в тестовом виде - и система лояльности: бонусы в виде баллов за покупку на сайте, а также несколько промокодов.

Проект был создан в учебных целях для лучшего понимания устройства веб-сайтов. В процессе создания веб-сайта я приобрела обширный опыт в разработке и интеграции различных технологий. Я освоила работу с Django REST Framework (DRF) для создания эффективных API, а также научилась взаимодействовать с API сторонних сервисов. Успешно интегрировала системы обработки электронных платежей Stripe и ЮKassa, что позволило реализовать работу с денежными операциями. Для управления аутентификацией пользователей использовала библиотеку Djoser. Я также поработала с асинхронными задачами, используя Celery, для повышения производительности, реализовала кеширование данных с помощью Redis и django-cachalot. Кроме того, я приобрела навыки контейнеризации приложений с использованием Docker, настройки веб-сервера Nginx, подготовки файлов для деплоя и самого деплоя приложения.

Надеюсь, этот опыт станет прочной основой для дальнейших профессиональных достижений и будущие проекты будут ещё более успешными и интересными!
